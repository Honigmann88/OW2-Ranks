[[[[[Run Guide]]]]]

1. Install node.js

2. Open OWO folder run-oneTime.bat

3. Type your discord bot token in index.js

4. Start the bot with Run-bot.bat (Windows) or Run-bot.sh (Linux and MacOS) and use "!sign xxxx" in a channel, which your leaderboards will stay.

5. Copy the random numbers you see in CMD. This is your channelID, type it in index.js

6. Close everything.

7. Start the bot with Run-bot.bat (Windows) or Run-bot.sh (Linux and MacOS)
